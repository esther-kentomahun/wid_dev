# Personal Bio Page

A minimalist, single-page personal biography website built from scratch using semantic HTML5 and custom styling. This project serves as a professional introduction, showcasing my academic background, evolving technical skill set, and core areas of specialty.

## 🚀 Live Demo
You can view the live project here: https://esther-kentomahun.github.io/wid_dev/

---

## 🎨 Design Philosophy
The page embraces a clean, editorial layout focused on strong typography and effective content hierarchy. By using a warm, natural background color (`burlywood`) and structured typography, the design presents professional information clearly and elegantly without relying on heavy imagery.

---

## 🛠️ Features & Technical Structure

- **Semantic HTML5:** Built using standard web structure elements (`<head>`, `<body>`, `<h2>`, `<ul>`, `<p>`) to ensure clean, readable, and accessible code.
- **Profile Image Styling:** Features a custom-cropped, circular profile image using inline CSS (`border-radius: 100%`).
- **Structured Highlights:** - **About Me:** A brief professional summary bridging a scientific background with modern web technologies.
  - **Areas of Specialty:** A targeted list highlighting core domains.
  - **Skills & Languages:** An organized breakdown of technical and soft skills.

---

## 💻 Technologies Used

* **HTML5:** For core page structure and text content.
* **CSS (Inline):** Used for foundational elements like the background color and circular image formatting.
